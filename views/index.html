Timestamp

Timestamp here

What is this ?

This is a simple back-end application providing a timestamp service .

What does that mean?

It means you can ping the server at the following endpoint:

[project_url]/api/timestamp/?date

And have it return a timestamp matching the selected date.

How does that look?

Sample request:

[project_url]/api/timestamp/1479663089000

Sample response:

{"unix": 1479663089000 ,"utc": "Sun, 20 Nov 2016 17:31:29 GMT"}

What about the button ?

We like a UI even for back-end projects, no matter how simple.
The button displays a sample request for date detailed in the input elements.
